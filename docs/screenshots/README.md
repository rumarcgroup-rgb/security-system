# Screenshot Placeholder Index

Place final manual screenshots in this folder using the filenames from `docs/website-user-manual-shot-list.md`.

## Capture Defaults
- Guard / employee screenshots: mobile-style viewport.
- Supervisor screenshots: desktop viewport.
- Admin screenshots: desktop viewport.

## Placeholder Workflow
1. Open the app in the correct role.
2. Navigate to the route listed in the shot list.
3. Set up the required UI state.
4. Save the screenshot with the exact filename listed in the shot list.
5. Replace any placeholder reference in `docs/website-user-manual.md` with the final image path.
